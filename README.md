# Flood Fill Project

This is a simple project I made using TypeScript to implement the Flood Fill algorithm. This algorithm is the same one used in paint programs for the "bucket fill" tool.

## What this project does:
The code takes a 2D array (an image), a starting point, and a new color. It then changes the color of that point and all connected points that have the same original color.

## Tools I used:
- TypeScript
- Recursion (to find the connected pixels)
- console.table (to show the result clearly in the terminal)

## How to run it:
First, make sure you have the files on your computer, then run these commands in the terminal:

1. npm install
2. npm run dani

## Project Files:
- index.ts: This is where I wrote the algorithm and the test data.
- package.json: Contains the scripts to compile and run the code.
